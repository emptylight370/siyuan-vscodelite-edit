# VSCode Lite Edit

从VSCode Lite脱胎而来，进一步美化界面。

使用暗色模式测试，预计明亮模式没有问题。

原作者：[TinkMingKing](https://github.com/TinkMingKing)

# 特性

- 各级标题样式（进一步调整）
- 微调布局及配色
- 个性化改动
- 表格显示效果

# 配置

主题使用 `import`导入不同的主题模块，可以通过调整注释修改显示效果。具体说明请查看[链接](https://github.com/lingfengyu-dreaming/siyuan-vscodelite-edit/Configure.md)。
